# Sales Collateral Creator

**Vertical:** Sales  
**Difficulty:** ⭐⭐⭐  
**Key Tools:** n8n · Canva API · Gamma/Beautiful.ai API · Notion · Google Drive · HubSpot  

---

## Use Case

Auto-generate all sales materials: product one-pagers, competitor battle cards, vertical-specific pitch decks, formatted case studies, brochures, FAQ sheets, and discovery call scripts. All branded, always up-to-date, stored in a centralized library.

---

## The Flow

Trigger: new product feature OR scheduled refresh → Pull updated product info from CRM/Notion → Generate materials: one-pager + battle card + pitch deck variant per vertical + case study format + FAQ → Canva API apply branding → Store in centralized Notion/Drive library → Notify sales team

---

## Real Client Example

A SaaS scale-up launching 3 new features/quarter. Before: 2 weeks to update all materials. After: 2h validation, everything auto-generated. Sales team always has the right version with the right arguments.
