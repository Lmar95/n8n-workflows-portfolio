# Content Repurposing Pipeline

**Vertical:** Marketing  
**Difficulty:** ⭐⭐⭐  
**Key Tools:** n8n · Descript · Canva API · Buffer · Taplio · Code  

---

## Use Case

Upload one long-form piece of content (webinar, podcast, YouTube video, blog post). Get back 15+ ready-to-publish assets across formats and platforms: Twitter threads, LinkedIn carousels, short posts, newsletter, infographic text, quotes, video clip timestamps.

---

## The Flow

Input long content (video/audio/text) → Descript transcription → AI extract key points + insights → Generate 15+ variants: thread, carousel, short posts, quotes, newsletter section, infographic text → Adapt tone per platform → Schedule via Buffer/Taplio (2-3 week calendar)

---

## Real Client Example

A consulting firm producing 1 webinar/month (1h runtime). Agent generates: 1 blog article, 5 LinkedIn posts, 3 Twitter threads, 10 quotes, 1 newsletter section. Total reach: 45K impressions vs 800 for the webinar alone.
