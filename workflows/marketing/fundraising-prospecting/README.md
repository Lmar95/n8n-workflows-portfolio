# Fundraising & Acquisition Prospecting

**Vertical:** Marketing  
**Difficulty:** ⭐⭐⭐  
**Key Tools:** Make · Crunchbase API · Dealroom API · Google Alerts · Clay · Dropcontact · Lemlist  

---

## Use Case

Monitor Crunchbase, Dealroom, and press releases for fresh fundraising rounds and acquisitions in real time. Reach out within 48–72h (the optimal window when companies are building their stack with new budget).

---

## The Flow

Crunchbase/Dealroom webhook or daily scrape → Filter by sector + raise size → Identify decision makers (CEO, CFO, new C-levels) → Clay enrich → Personalize outreach mentioning the raise + budget timing → Lemlist sequence launch within 48h

---

## Real Client Example

A cybersecurity consulting firm detected a fintech raising 15M€. Contacted within 24h: 'Congratulations on your Series A! With your growth, security becomes critical...' Meeting booked with the CTO, 120K€ contract signed.
