# Performance Review Prep

**Vertical:** HR  
**Difficulty:** ⭐⭐⭐  
**Key Tools:** n8n · HubSpot/BambooHR · Notion · Google Sheets · Google Docs · Code  

---

## Use Case

For each upcoming annual review, automatically compile all relevant data: annual objectives and achievement rate, KPIs reached, 360° feedback received, completed projects, training followed, notable incidents and successes. Pre-fill manager dossier with suggested discussion points + employee self-assessment form.

---

## The Flow

Upcoming review detected (D-14) → Pull from all systems: OKRs progress + KPI data + peer feedback + projects completed + training records + support interactions → AI compile structured dossier for manager: performance summary + suggested discussion points + areas to explore → Generate employee self-assessment form → Send both D-7

---

## Real Client Example

A manager with 8 direct reports spending 16h preparing annual reviews (collecting scattered information). With automation: 2h of review only. Full focus on conversation quality. Team satisfaction with evaluation process: +32 points.
