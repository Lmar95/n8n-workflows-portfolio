# Job Offer Generator

**Vertical:** HR  
**Difficulty:** ⭐⭐  
**Key Tools:** n8n · Notion · LinkedIn API · Welcome to the Jungle API · Code  

---

## Use Case

Give the job title and a few bullet points on the ideal profile. Get a complete, SEO-optimized job posting: engaging company description, detailed missions, required profile, technical skills, soft skills, benefits, and recruitment process. 2–3 variants adapted to your employer brand tone.

---

## The Flow

Input: job title + profile notes → AI generate full posting: SEO headline + company pitch + missions + profile + skills + benefits + process → Generate 2-3 variants (classic / modern / startup tone) → Push to ATS → Publish to LinkedIn + Welcome to the Jungle → Notify hiring manager

---

## Real Client Example

A scale-up recruiting 15 positions/quarter. Before: HR overwhelmed, postings published 2 weeks after validation. After: postings generated in 15 minutes, published same day. Time-to-hire reduced by 12 days.
