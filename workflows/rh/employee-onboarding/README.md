# Employee Onboarding

**Vertical:** HR  
**Difficulty:** ⭐⭐⭐  
**Key Tools:** n8n · Notion/Asana · Google Workspace API · Slack · Typeform · Loom · HubSpot  

---

## Use Case

New hire signs contract = automated 30–90 day onboarding journey: account provisioning, equipment ordering, training schedule, team introductions, required readings, first projects, regular manager check-ins. Progress tracked, blockers detected, D+30 and D+90 feedback collected.

---

## The Flow

Offer accepted in ATS → Trigger: create all accounts (email, Slack, tools) → Schedule: equipment delivery + badge + welcome package → Day 1: send welcome Loom video + onboarding checklist → Week 1-2: daily micro-tasks + buddy assignment → Weekly: manager check-in prompt → D+30: satisfaction survey → D+90: full feedback form

---

## Real Client Example

A tech company with 25 hires/year had 30% first-year turnover (chaotic onboarding). Automated structured journey: turnover at 8%, onboarding NPS of 78. Recruiting savings: 150K€/year.
